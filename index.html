Stack Overflow Friends The Stack Overflow Friends app you were waiting for!

Create a list of friends from Stack Overflow by searching for a name or user ID. Add them to your list and keep track of their stats. Reputation, badges, and activity is always refreshed when you open the app.

Currently the app is available in english, german, and spanish.
Legal
Privacy Policy for Stack Overflow Friends iOS app The Stack Overflow Friends iOS app uses public data from the Stack Exchange API.

It stores the IDs of users that were added to the friend list on the phone, and only on the phone, in order to remember the added friends and load them the next time the app is opened.

All other information that is consumed via the Stack Exchange API is held in memory, only as long as the app is running.

No data is stored on any remote server.
